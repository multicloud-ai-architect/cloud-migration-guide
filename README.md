## Overview

On-premise to multi-cloud migration reference — 7R patterns, SQL Server to RDS, IaaS/PaaS/SaaS decision guide, ADRs. This project is part of the MAIA — Multi-Cloud AI Architect portfolio, providing reference architectures and reusable building blocks for cloud architects working across AWS, Azure, and GCP.
